
# Versions
* React: 18.2.0
* Node: 18.7.0
* npm: 8.15.0

# How to Start:

## Step 1:
Go to each **server** and **todo** folder and run this below command in CMD

    npm install

## Step 2: 
Open two differnt terminals for both **todo** and **server** and run this command

* todo: 

        npm start
* server: 

        npm start
